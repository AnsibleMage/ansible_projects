# 📡 Ansible 프로젝트 중심 허브

> **"Ansible 멀티버스 내의 모든 발현을 위한 우주적 교차점."**

**Ansible Station**의 중앙 저장소에 오신 것을 환영합니다. 이곳은 로블록스(Roblox)와 웹(Web) 전반에 걸친 고성능 개발 인프라를 총괄합니다. 본 허브는 **VCR (Verify-Code-Refactor)** 방법론과 **Antigravity AI** 오케스트레이션에 기반한 프로젝트들의 설계적 토대 역할을 합니다.

## 🎮 로블록스 개척지 (`/1000_Roblox`)

로블록스 섹터는 정밀 물리 엔진, 3D 월드 빌딩 및 고급 Luau 스크립팅에 집중합니다.

### ✅ [100_Ansible Logic Jump](./1000_Roblox/100_Ansible%20Logic%20Jump) - **LIVE** 🟢
*   **장르**: 극한 고공 점프 / 타임 어택
*   **핵심 기술**: Rojo, DataStore, 정밀 타이머
*   **상태**: 런칭 완료 (2026-01-25)

### 🔄 [200_Roblox_Fly_paper_plane](./1000_Roblox/200_Roblox_Fly_paper_plane) - **개발 중** 🟡
*   **장르**: 1인칭 비행 시뮬레이션
*   **핵심 기술**: BodyVelocity 물리, 커스텀 비행 카메라
*   **상태**: 탑승 로직 디버깅 중 (2026-01-26)

본 저장소의 모든 프로젝트는 엄격한 개발 표준을 따릅니다:

*   **[VCR 방법론](./1000_Roblox/doc/Roblox_Dev%20Methodology/203_VCR_개발방법론_v1.1.md)**: **Verify-Code-Refactor** 사이클을 통한 마찰 없는 생산성.
