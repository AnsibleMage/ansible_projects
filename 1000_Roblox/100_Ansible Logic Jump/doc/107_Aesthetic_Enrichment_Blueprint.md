# 107_Aesthetic_Enrichment_Blueprint: 프리미엄 숲속 환경 명세서

## 1. 개요 (Overview)
기존의 단순 파트 기반의 조경이 사용자에게 실망을 안겨주었습니다. 본 문서는 로블록스의 엔진 성능을 최대한 활용하여 '프리미엄'하고 '몰입감' 있는 숲속 공터의 분위기를 재창조하기 위한 미학적 파라미터를 정의합니다.

## 2. 고급 라이팅 시스템 (Advanced Lighting)
단순한 색상 변경이 아닌, 대기 효과(Atmospheric Effects)를 주입합니다.

| Effect | Property | Value | Note |
| :--- | :--- | :--- | :--- |
| **Atmosphere** | Density | `0.3` | 원경의 몽환적인 안개 효과 |
| **Atmosphere** | Haze | `2.0` | 빛의 산란 극대화 |
| **Bloom** | Intensity | `1.0` | 네온 및 태양광의 부드러운 번짐 |
| **SunRays** | Intensity | `0.2` | 나뭇잎 사이로 비치는 햇살(God-rays) |

## 3. 유기적 모델링 가이드 (Organic Modeling)
- **Trunk**: 단일 파트가 아닌, 밑동이 더 굵고 위로 갈수록 얇아지는 다중 파트 구성.
- **Leaves**: 하나의 공 모양이 아닌, 여러 개의 `LeafBlock`을 불규칙하게 겹쳐 풍성한 질감 표현.
- **Color**: `LeafGreen` 한 가지가 아닌, `DarkGreen`, `ForestGreen`, `SlimeGreen`을 섞어 깊이감 형성.

## 4. 트랙 가시성 개선
- **Start/Finish**: 발광 강도를 높이고, 하단에 부드러운 파티클 효과(선택 사항)를 고려.
- **Track**: 잔디와 더 잘 어우러지는 `Mud` 또는 `Sand` 재질의 경계선 추가.
