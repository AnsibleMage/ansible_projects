# 106_프로그램별 계정 필요성 및 클라우드 서비스 가이드

## 1. 개요
로블록스 개발 환경을 구축하면서 각 프로그램(Roblox Studio, VS Code 등)에서 계정이 왜 필요한지, 그리고 어떤 계정을 준비해야 하는지 안내합니다.

## 2. 프로그램별 계정 상세

### [Roblox 계정 (필수)]
*   **용도**: 게임 게시(Publish), 에셋 업로드, 크리에이터 대시보드 접근, 수익 창출.
*   **필요성**: 로블록스 스튜디오는 계정 로그인 없이는 게임을 서버에 저장하거나 멀티플레이 테스트를 할 수 없습니다. 
*   **참고**: 이미 로그인해 두셨다고 하니 이 부분은 준비가 완료되었습니다.

### [GitHub 계정 (강력 권장)]
*   **용도**: 소스 코드 버전 관리(Git), 안티그래비티와의 코드 공유, 프로젝트 백업.
*   **필요성**: 
    1.  **안전성**: 코드를 날렸을 때 이전 상태로 복구할 수 있습니다.
    2.  **협업**: 나중에 다른 개발자나 AI와 더 긴밀하게 협업할 때 표준이 됩니다.
*   **안티그래비티와 연동**: 현재 `ansible_config` 등 GitHub 리포지토리를 사용 중이시므로, 작업물을 여기에 푸시(Push)하려면 계정이 필요합니다.

### [VS Code (선택 사항)]
*   **용도**: 설정 동기화, 확장 프로그램 관리.
*   **필요성**: 로그인하지 않아도 코딩에는 지장이 없습니다. 다만, **GitHub Copilot** 등의 유료 AI 보조 도구를 사용하려면 GitHub 계정으로 로그인이 필요합니다.
*   **참고**: 현재 안티그래비티와 함께 작업하신다면 VS Code 자체 로그인은 필수는 아닙니다.

### [Discord 계정 (권장)]
*   **용도**: 로블록스 개발자 커뮤니티(DevForum 등) 및 각종 플러그인(Rojo 등) 지원 채널 이용.
*   **필요성**: 최신 트렌드를 파악하고 기술적인 도움을 받기에 가장 좋은 플랫폼입니다.

## 3. 요약 및 권장 사항
1.  **Roblox**: 로그인 유지 (완료)
2.  **GitHub**: VS Code에서 로그인 권장 (버전 관리 및 백업용)
3.  **VS Code**: 기능적으로는 필수 아님 (선택)

## 4. 안티그래비티의 조언
사용자님, 현재 `ansible_config`를 통해 저와 소통하고 계시므로 **GitHub 계정**은 반드시 연동해 두시는 것을 추천드립니다. 이를 통해 제가 작성한 문서와 코드를 안전하게 관리하고 전 세계 어디서든 이어서 작업하실 수 있습니다.
