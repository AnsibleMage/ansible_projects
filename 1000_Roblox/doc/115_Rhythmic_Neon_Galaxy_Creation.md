# 115_바이브 코딩: 리듬 네온 갤럭시(Rhythmic Neon Galaxy) 창조

## 1. 개요
성공적인 첫 연결을 기념하여, 이제 단순한 객체 생성을 넘어 **로직(Logic)**이 살아있는 공간을 창조합니다. 10개의 네온 발판이 은하계의 길처럼 배치되며, 각 발판은 안시블의 비트에 맞춰 생동감 있게 반응합니다.

## 2. 현현 상세 (Manifestation Details)

### [객체 리스트]
*   **이름**: `NeonPlatform_1` ~ `NeonPlatform_10`
*   **배치**: Z축 방향으로 15스터드 간격으로 일렬 배치 (성장하는 점프맵의 기초)
*   **비주얼**: `Neon` 재질, 초기 색상 `Medium Royal Blue`

### [리듬 스크립트: RhythmScript]
*   **위치**: `ServerScriptService`
*   **로직**:
    1. `Workspace` 내의 `NeonPlatform_`으로 시작하는 모든 객체를 탐색합니다.
    2. 무한 루프(`while true do`)를 통해 0.5초 간격으로 색상을 순환시킵니다.
    3. 색상 시퀀스: `Cyan` → `Electric Purple` → `Neon Orange`

## 3. 실행 로그 (안티그래비티 수행)
1.  **신규 파일 생성**: `src/server/RhythmScript.server.lua`에 루아 코드 작성.
2.  **프로젝트 동기화**: `default.project.json`을 업데이트하여 10개 객체의 위상 정의.
3.  **Rojo Sync**: 실시간으로 로블록스 스튜디오에 정적 객체와 동적 스크립트 주입.

## 4. 확인 방법
1.  로블록스 스튜디오 상단의 **Play** 버튼을 누릅니다.
2.  캐릭터가 소환되면 눈앞에 펼쳐진 10개의 빛나는 발판들을 확인하세요.
3.  발판들이 살아있는 것처럼 색을 바꾸며 사용자님을 기다리고 있을 것입니다.
