# 112_로블록스 게임 설정(Game Settings) 및 HTTP 권한 활성화 가이드

## 1. 개요
보내주신 스크린샷을 분석한 결과, **'Studio Settings(전역 설정)'** 창에서 검색하고 계신 것을 확인했습니다. 하지만 'HTTP 요청 허용' 설정은 프로젝트별 **'Game Settings(게임 설정)'**에 위치하며, 이를 위해서는 먼저 게임을 로블록스 클라우드에 **게시(Publish)** 해야 합니다.

## 2. 정확한 설정 단계

### [1단계: 게임 게시 (Publish to Roblox)]
'Game Settings' 메뉴를 활성화하기 위해 먼저 수행해야 합니다.
1.  상단 메뉴에서 **File > Publish to Roblox**를 클릭합니다.
2.  게재할 게임의 이름(예: `Neon Galaxy Obby`)을 입력하고 **Create**를 누릅니다.

### [2단계: 게임 설정(Game Settings) 진입]
게시가 완료되면 이제 프로젝트별 설정을 바꿀 수 있습니다.
1.  상단 **Home** 탭을 클릭합니다.
2.  오른쪽 상단 근처에 있는 **Game Settings** (톱니바위 아이콘) 버튼이 활성화됩니다. 이를 클릭합니다.
    *   *전체 Studio 설정과는 다른, 이 프로젝트 전용 설정창이 열립니다.*

### [3단계: HTTP 요청 허용 활성화]
1.  왼쪽 메뉴에서 **Security** 항목을 선택합니다.
2.  **Allow HTTP Requests** 항목을 찾아서 **On**으로 활성화합니다.
3.  **Save** 버튼을 눌러 적용합니다.

## 3. 그림으로 보는 위치 가이드
*사용자님의 화면 상단 파란색 **Collaborate** 버튼 바로 왼쪽 근처에 **Game Settings** 아이콘이 생기게 됩니다.*

## 4. 안티그래비티의 조언
이 설정이 완료되어야 제가 외부에서 `npx robloxstudio-mcp`를 통해 스튜디오 내부로 명령을 전달할 수 있습니다. 

게시(Publish)를 먼저 진행하신 후, **Security** 탭에서 설정을 다시 시도해 주세요!
