# 102_로블록스 개발 환경 구축 및 AI(안티그래비티) 연동 가이드

## 1. 개요
본 문서는 MacBook Pro 환경에서 로블록스 게임 개발을 효율적으로 수행하기 위한 필수 프로그램 설치와, 안티그래비티(AI 에이전트)를 개발 프로세스에 깊숙이 연동하여 '바이브 코딩'을 실현하는 방법을 안내합니다.

## 2. 필수 프로그램 설치 (Setup)
로블록스 스튜디오의 한계를 넘어 외부 에디터(VS Code)와 AI를 연동하기 위한 '전문가용 워크플로우'를 구축합니다.

### [기본 도구]
1.  **Roblox Studio**: 게임 렌더링 및 최종 퍼블리싱용 (로블록스 공식 웹사이트 설치)
2.  **Visual Studio Code (VS Code)**: 메인 코드 에디터 (AI 연동의 핵심)

### [연동 도구 - Rojo (로조)]
Rojo는 VS Code에서 쓴 코드를 실시간으로 Roblox Studio로 전송해주는 핵심 도구입니다.
*   **설치 방법**: 
    1.  VS Code 내 확장 프로그램 상점에서 **'Rojo'** 검색 및 설치
    2.  [Rojo 공식 릴리즈](https://github.com/rojo-rbx/rojo/releases)에서 macOS용 바이너리 다운로드 또는 `aftman`을 통한 설치

### [코드 보조 도구]
*   **Roblox LSP**: Luau 언어의 자동 완성 및 오류 검사 기능을 VS Code에 제공합니다.
*   **Luau Language Server**: 고성능 타입 체크 및 인텔리센스 제공.

## 3. 안티그래비티(AI) 연동: MCP 및 Vibe Coding 설정
로블록스 스튜디오 내부의 AI 어시스턴트 외에, 안티그래비티와 프로젝트를 직접 연결하는 고도화된 방식입니다.

### [MCP (Model Context Protocol) 서버 활용]
안티그래비티와 같은 외부 AI 모델이 사용자의 로블록스 프로젝트 계층 구조를 직접 읽고 수정하게 만드는 프로토콜입니다.
1.  **Roblox Studio MCP Server 설치**: 
    *   Node.js/TypeScript 기반의 오픈소스 MCP 서버를 실행합니다.
    *   로블록스 스튜디오 내에 **'Rojo Plugin'**과 **'HTTP Bridge Plugin'**을 설치합니다.
2.  **연동 효과**:
    *   안티그래비티가 `task.md`를 보며 직접 로블록스 파일 구조를 탐색 (`Explore hierarchy`)
    *   바이브 코딩: "메인 로비에 네온 스타일 발판을 10개 생성하고 리듬 스크립트를 주입해줘"라고 지시하면 안티그래비티가 직접 오브젝트 생성 및 스크립팅 수행

## 4. 권장 워크플로우 (Vibe Coding Logic)
1.  **Rojo 실행**: VS Code에서 Rojo 서버를 켜고, Roblox Studio에서 연결합니다.
2.  **의도 전달 (Vibe)**: 사용자가 안티그래비티에게 `101_` 기획서 내용을 바탕으로 구체적인 기능을 요청합니다.
3.  **코드 생성 및 동기화**: 안티그래비티가 VS Code에 `.luau` 파일을 생성하면, Rojo가 이를 즉시 Roblox Studio 게임 엔진에 반영합니다.
4.  **실시간 피드백**: 게임 실행(Play)을 통해 결과를 확인하고, 안티그래비티와 대화하며 미세 조정을 진행합니다.
