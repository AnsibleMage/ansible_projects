웹 브라우저에서 아래 링크를 클릭하고 **"Install"** 버튼을 누르면 로블록스 스튜디오에 자동으로 설치됩니다.

*   **MCP Server Plugin**: [설치 링크](https://create.roblox.com/store/asset/75577477776988)

*   **경로**: `/Users/changjaeyou/Documents/Roblox/Plugins/MCPPlugin.rbxmx`

## 3. 설치 확인 및 연동
1.  **Roblox Studio**를 완전히 종료 후 재실행합니다.
2.  상단 메뉴의 **Plugins** 탭을 클릭합니다.
3.  **Rojo** 아이콘과 **MCP Server** 아이콘이 나타났는지 확인합니다.
4.  각 아이콘을 클릭하여 활성화(Connect 또는 Run)합니다.
