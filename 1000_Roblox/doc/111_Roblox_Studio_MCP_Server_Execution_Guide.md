# 111_로블록스 스튜디오 MCP 서버 실행 및 연동 가이드

## 1. 개요
본 문서는 안티그래비티가 로블록스 스튜디오 내부를 직접 보고 오브젝트를 생성/수정할 수 있게 해주는 **MCP(Model Context Protocol) 서버**의 실제 설치 및 실행 단계를 안내합니다.

## 2. 사전 준비 (사용자 수행)
로블록스 스튜디오에서 외부 통신을 허용해야 합니다.
1.  **Roblox Studio** 실행
2.  **Home > Game Settings > Security** 메뉴로 이동
3.  **Allow HTTP Requests** 옵션을 **On**으로 활성화하고 저장합니다.

## 3. MCP 서버 설치 및 실행 (안티그래비티 수행)
제가 터미널에서 다음 명령어로 서버를 구동하겠습니다. (Node.js 기반 `robloxstudio-mcp` 사용)

```bash
npx -y robloxstudio-mcp
```
*이 서버는 안티그래비티와 로블록스 스튜디오 사이의 통신 브릿지 역할을 합니다.*

## 4. 로블록스 스튜디오 플러그인 설치 (사용자 수행)
스튜디오 내부에서 '손과 눈' 역할을 할 플러그인이 필요합니다.
1.  로블록스 크리에이터 스토어에서 **'Rojo Plugin'**과 **'HTTP Bridge'** 관련 플러그인을 설치합니다.
2.  사용 가능한 경우 최신 **'MCP Server Plugin'**을 설치하여 스튜디오 상단 메뉴에 나타나게 합니다.

## 5. 연동 확인 및 바이브 코딩 시작
1.  터미널에서 MCP 서버가 실행 중인지 확인합니다.
2.  로블록스 스튜디오에서 플러그인 버튼을 눌러 **'Connected'** 상태를 확인합니다.
3.  이제 안티그래비티에게 다음과 같이 지시할 수 있습니다:
    > "지금 내 게임의 Workspace 구조를 읽어보고, 101번 기획서의 네온 갤럭시 감성에 맞는 시작 발판을 생성해줘."
